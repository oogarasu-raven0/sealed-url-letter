# 概要

URLベースで手紙を送信するサイトです

1. ブラウザでサイトを開いて内容を記入
2. パスワードを決めてURLを生成
3. パスワードとURLを共有
4. もらった人はURLを開いてパスワードを入力する
5. 内容を確認する

の流れになっています

DBレスの構成なので、URLが漏れない限りはセキュアです

一応、XSS対策も入れています
